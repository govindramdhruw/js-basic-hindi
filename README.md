# js-basic-hindi
This repo is for learning javascript from basic.

# introduction of js
JavaScript ek web development language hai jo websites ko interactive banane ke liye istemal hoti hai. HTML aur CSS ke saath milakar use hoti hai. Isme elements ko manipulate kar sakte hain, animations create kar sakte hain, aur user input handle kar sakte hain.

# variables and data types
Variables JavaScript mein values ko store karne ke liye use hoti hain. Har variable ek naam se identify hoti hai. Data types determine karte hain ki variable mein kis tarah ki value store hogi.

1. String:\n
String ek series of characters hai, jo single quotes (' ') ya double quotes (" ") mein hoti hai.
Example:
```javascript
let name = "John Doe";
```

2. Number
Number ek numeric value hai.
Example:
```javascript
let age = 25;
```

3. Boolean ek true ya false value hai.
Example:
```javascript
let isStudent = true;
```

4. Array
Array ek collection of values hai, jo comma-separated brackets [] ke andar hote hain.
Example:
```javascript
let fruits = ["apple", "banana", "orange"];
```

5. Object
Object ek collection of key-value pairs hai.
Example:
```javascript
let person = { name: "John", age: 30, isStudent: true };
```

6. Undefined
Undefined data type hota hai jab koi variable declare kiya jata hai par usme koi value assign nahi hoti.
Example:
```javascript
let city;
```

7. Null
Null data type ek special value hai, jo ek variable ko empty ya null value ke roop mein set karta hai.
Example:
```javascript
let result = null;
```

8. BigInt
BigInt data type mein hum bahut bade integers ko represent kar sakte hain, jo Number data type ki range se bahar ho.
Example:
```javascript
let bigNumber = 123456789012345678901234567890n;
```
