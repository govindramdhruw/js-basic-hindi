# introduction of js
JavaScript ek web development language hai jo websites ko interactive banane ke liye istemal hoti hai. HTML aur CSS ke saath milakar use hoti hai. Isme elements ko manipulate kar sakte hain, animations create kar sakte hain, aur user input handle kar sakte hain.
